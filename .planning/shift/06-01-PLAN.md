# Phase 6 — AI Reschedule

## Objective

Implement AI-powered task rescheduling. User taps Reschedule (or edits AI fields on a task), all incomplete tasks get sent to an Edge Function that calls OpenRouter via model fallback chain. AI returns optimally arranged tasks with justifications. User can undo within 5 seconds. Toast with undo at top of screen.

## Context

- Phases 1–5 complete: project structure, auth, Sentry, schedule UI, task CRUD all working
- `RescheduleSheet.tsx` exists — CTA is currently a no-op
- `TaskFormSheet.tsx` exists — "Let AI decide" toggle is disabled/forced off
- `src/services/ai.ts` exists — placeholder `rescheduleTasks` function
- `src/store/` is empty — need to create undo store
- TanStack Query already set up with `useTasks`, `useCreateTask`, `useUpdateTask` hooks
- Supabase Edge Functions not yet created — need Supabase CLI
- OpenRouter API key not yet configured
- User preferences not implemented yet (Phase 8) — pass empty string for `userContext`
- No `Toast` component exists yet — build from scratch

**Key decisions:**
- **TDD approach** — write tests first, then implement to make them pass
- **OpenAI SDK** in Edge Function — cleaner code, Zod-validated structured output, automatic JSON parsing
- Single Edge Function file (small project, keeps it simple)
- All incomplete tasks (any date) sent to Edge Function
- Tasks with `aiDecidesTime: true` get time assigned immediately on creation (sheet shows spinner, closes when AI responds)
- Reschedule triggers: Reschedule button, task creation with `aiDecidesTime: true`, task edit where AI fields changed
- Toast: top of screen, 5s auto-dismiss, fade animation, tap "Undo" only
- Undo: full snapshot restore (all tasks reverted, written to Supabase)

```ts
import { create } from 'zustand';
import type { Task } from '@/types/task';

type UndoState = {
  snapshot: Task[] | null;
  timeoutId: ReturnType<typeof setTimeout> | null;
  setSnapshot: (tasks: Task[]) => void;
  clearSnapshot: () => void;
  startTimeout: (onExpire: () => void, ms?: number) => void;
  cancelTimeout: () => void;
};

export const useUndoStore = create<UndoState>((set, get) => ({
  snapshot: null,
  timeoutId: null,
  setSnapshot: (tasks) => set({ snapshot: tasks }),
  clearSnapshot: () => set({ snapshot: null }),
  startTimeout: (onExpire, ms = 5000) => {
    const existing = get().timeoutId;
    if (existing) clearTimeout(existing);
    const id = setTimeout(() => {
      onExpire();
      set({ snapshot: null, timeoutId: null });
    }, ms);
    set({ timeoutId: id });
  },
  cancelTimeout: () => {
    const id = get().timeoutId;
    if (id) clearTimeout(id);
    set({ timeoutId: null });
  },
}));
```

export async function fetchIncompleteTasks(): Promise<Task[]> {
  const { data, error } = await supabase
    .from('tasks')
    .select('*')
    .eq('completed', false)
    .order('start_time', { ascending: true });

  if (error) throw new Error(error.message);
  return mapTasks(data ?? []);
}

export async function batchUpdateTasks(
  updates: {
    id: string;
    startTime: string;
    endTime: string;
    durationMinutes: number;
    aiJustification: string;
    aiContext: string;
  }[]
): Promise<Task[]> {
  const results: Task[] = [];
  for (const update of updates) {
    const { data, error } = await supabase
      .from('tasks')
      .update({
        start_time: update.startTime,
        end_time: update.endTime,
        duration_minutes: update.durationMinutes,
        ai_justification: update.aiJustification,
        ai_context: update.aiContext,
      })
      .eq('id', update.id)
      .select()
      .single();

    if (error) throw new Error(error.message);
    results.push(mapTask(data));
  }
  return results;
}

import { supabase } from '@/services/supabase';
import type { Task } from '@/types/task';

export type RescheduleResult = {
  id: string;
  startTime: string;
  endTime: string;
  durationMinutes: number;
  aiJustification: string;
  aiContext: string;
};

export async function rescheduleTasks(
  tasks: Task[],
  userContext: string,
  whatChanged: string,
): Promise<RescheduleResult[]> {
  const { data: { session } } = await supabase.auth.getSession();
  if (!session) throw new Error('Not authenticated');

  const supabaseUrl = process.env.EXPO_PUBLIC_SUPABASE_URL;
  if (!supabaseUrl) throw new Error('Missing EXPO_PUBLIC_SUPABASE_URL');

  const response = await fetch(
    `${supabaseUrl}/functions/v1/reschedule`,
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${session.access_token}`,
      },
      body: JSON.stringify({
        tasks: tasks.map(t => ({
          id: t.id,
          name: t.name,
          startTime: t.startTime,
          endTime: t.endTime,
          durationMinutes: t.durationMinutes,
          deadline: t.deadline,
          aiContext: t.aiContext,
          aiDecidesTime: t.aiDecidesTime,
        })),
        userContext,
        whatChanged,
      }),
    },
  );

  if (!response.ok) {
    const error = await response.text();
    throw new Error(`Reschedule failed: ${error}`);
  }

  const result = await response.json();
  if (!Array.isArray(result.tasks)) {
    throw new Error('Invalid response: tasks is not an array');
  }
  return result.tasks as RescheduleResult[];
}
```

import { useToast } from '@/providers/toast-provider';

export function useReschedule() {
  const queryClient = useQueryClient();
  const toast = useToast();
  const undoStore = useUndoStore();

  const mutation = useMutation({
    mutationFn: async ({ whatChanged }: { whatChanged: string }) => {
      const tasks = await api.fetchIncompleteTasks();
      undoStore.setSnapshot(tasks);
      const result = await rescheduleTasks(tasks, '', whatChanged);
      return api.batchUpdateTasks(result);
    },
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['tasks'] });
      toast.show({
        message: 'Schedule updated',
        actionLabel: 'Undo',
        onAction: handleUndo,
        duration: RESCHEDULE_CONSTANTS.TOAST_DURATION_MS,
      });
      undoStore.startTimeout(
        () => undoStore.clearSnapshot(),
        RESCHEDULE_CONSTANTS.UNDO_TIMEOUT_MS,
      );
    },
    onError: () => {
      undoStore.clearSnapshot();
    },
  });

  async function handleUndo() {
    const snapshot = undoStore.snapshot;
    if (!snapshot) return;
    undoStore.cancelTimeout();
    await api.batchUpdateTasks(
      snapshot.map(t => ({
        id: t.id,
        startTime: t.startTime,
        endTime: t.endTime,
        durationMinutes: t.durationMinutes,
        aiJustification: t.aiJustification ?? '',
        aiContext: t.aiContext ?? '',
      }))
    );
    queryClient.invalidateQueries({ queryKey: ['tasks'] });
    undoStore.clearSnapshot();
    toast.hide();
  }

  return mutation;
}
```

#### Step 2: Create handler

**File:** `supabase/functions/reschedule/index.ts`

```ts
import "jsr:@supabase/functions-js/edge-runtime.d.ts";
import OpenAI from "npm:openai";
import { z } from "npm:zod";
import { zodResponseFormat } from "npm:openai/helpers/zod";
import {
  MODELS,
  OPENROUTER_BASE_URL,
  TIMEOUT_MS,
  MAX_TOKENS,
  SYSTEM_PROMPT,
  SCHEMA_NAME,
} from "./config.ts";

// --- Zod schema for structured output ---
const TaskResultSchema = z.object({
  id: z.string(),
  startTime: z.string(),
  endTime: z.string(),
  durationMinutes: z.number(),
  aiJustification: z.string(),
  aiContext: z.string(),
});

const RescheduleSchema = z.object({
  tasks: z.array(TaskResultSchema),
});

// --- Handler ---
Deno.serve(async (req) => {
  try {
    const authHeader = req.headers.get("Authorization");
    if (!authHeader) {
      return new Response(JSON.stringify({ error: "Missing authorization" }), { status: 401 });
    }

    const body = await req.json();
    const { tasks, userContext, whatChanged } = body;

    if (!Array.isArray(tasks) || tasks.length === 0) {
      return new Response(JSON.stringify({ error: "tasks array required" }), { status: 400 });
    }

    const today = new Date().toISOString().slice(0, 10);
    const systemPrompt = SYSTEM_PROMPT.replace("{today}", today);

    const userMessage = `Tasks:\n${JSON.stringify(tasks, null, 2)}\n\nUser context: ${userContext || "None provided"}\n\nWhat changed: ${whatChanged || "Initial scheduling"}`;

    const openai = new OpenAI({
      baseURL: OPENROUTER_BASE_URL,
      apiKey: Deno.env.get("OPENROUTER_API_KEY"),
    });

    let lastError: Error | null = null;

    for (const model of MODELS) {
      try {
        const completion = await openai.beta.chat.completions.parse({
          model,
          messages: [
            { role: "system", content: systemPrompt },
            { role: "user", content: userMessage },
          ],
          response_format: zodResponseFormat(RescheduleSchema, SCHEMA_NAME),
          max_tokens: MAX_TOKENS,
        });

        const parsed = completion.choices[0].message.parsed;
        if (!parsed) {
          throw new Error("Model returned null parsed output");
        }

        const inputIds = new Set(tasks.map((t: { id: string }) => t.id));
        const outputIds = new Set(parsed.tasks.map(t => t.id));
        for (const id of inputIds) {
          if (!outputIds.has(id)) {
            throw new Error(`Missing task id in output: ${id}`);
          }
        }

        return new Response(JSON.stringify({ tasks: parsed.tasks }), {
          headers: { "Content-Type": "application/json" },
          status: 200,
        });
      } catch (err) {
        lastError = err instanceof Error ? err : new Error(String(err));
        console.error(`Model ${model} failed:`, lastError.message);
        continue;
      }
    }

    return new Response(
      JSON.stringify({ error: `All models failed. Last error: ${lastError?.message}` }),
      { status: 500, headers: { "Content-Type": "application/json" } }
    );
  } catch (err) {
    console.error("Edge Function error:", err);
    return new Response(
      JSON.stringify({ error: err instanceof Error ? err.message : "Unknown error" }),
      { status: 500, headers: { "Content-Type": "application/json" } }
    );
  }
});
```

## Notes

- **TDD:** Tests are written first in each task. Implementation follows. Run tests after each task to verify.

  Change these files to swap models, edit prompts, or tune timings. No other files need updating.
- **OpenAI SDK in Edge Function:** Uses `npm:openai` and `npm:zod` for structured output. `zodResponseFormat` ensures the AI returns valid JSON matching our schema. If a model doesn't support structured output, the SDK throws and we fall back to the next model.

- **Toast position** — top of screen, below safe area. Won't conflict with FAB (bottom-right).
- **Undo timeout** — 5 seconds. Cleared on component unmount.
- **Edge Function timeout** — 30s per model attempt. Total worst case: ~90s. Client shows loading state.
- **Reschedule on edit** — only triggers if `aiDecidesTime` or `aiContext` actually changed.
- **Edge Function auth** — validates Supabase JWT from `Authorization` header.
- **Deno imports** — `npm:` prefix for npm packages, `jsr:` for Deno standard library.
